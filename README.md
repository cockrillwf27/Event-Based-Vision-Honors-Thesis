## Overview

This thesis explores the use of event-based (neuromorphic) vision sensors as a non-invasive method for real-time monitoring of legacy control interfaces, analog gauges, and human-machine interactions. 

Event-based cameras only generate data when changes occur in a scene, offering major advantages over traditional RGB cameras in latency, power consumption, and performance under varying lighting. This work investigates how these sensors can provide efficient, trustworthy visual observability in environments where direct hardware integration is impractical or prohibited (e.g., certified systems, black-box equipment, or legacy infrastructure).

## Research Question

How can event-based neuromorphic vision sensors be utilized and implemented across multiple applications to provide efficient, non-invasive, real-time observability of dynamic physical interfaces, human actions, and high-speed processes — particularly where human-in-the-loop monitoring is currently required?
